-Place this "code" folder anywhere on your computer.  
-Download the NetBeans IDE from http://www.netbeans.org.  Get either the "Java EE" version or the "All"/"Full" version.
-Launch NetBeans and select File->Open Project...
-Select the "code" folder.  It will open as the project "StripesByExample"
-Right click on the project and select "Run..." to get started.
